# Traffic-Lights

## How to use?
  - download project
  - open project folder
  - open CMD in the project folder
  - type next commands(must have node installed):
  
  ```
  npm install
  ```
  ```
  npm run dev
  ```
  


## I recommend to download the project and open index.html with your browser from /src/
